Frequently Asked Questions

How big can my file be?

You can upload files up to 30GB per transfer. For larger files, you can split them into multiple uploads.

Is it secure?

Yes! Files are encrypted with 256-bit AES encryption both during transfer and while stored on our servers. We use bank-level security protocols.

How long are files stored?

Files are automatically deleted after 30 days. You can also set custom expiry dates when uploading.

Do recipients need an account?

No! Recipients only need the download link. No registration or account creation required.

Can I protect my file with a password?

Yes! You can add an optional password when uploading. Recipients will need this password to download the file.

Can I pause/resume uploads?

Yes! Our upload system supports resumable uploads. If your connection drops, you can continue from where you left off.
